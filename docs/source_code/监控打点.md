# 监控打点

feapder 内置 InfluxDB 1.x 监控打点，配合 Grafana 可视化。监控数据默认保留 180 天，滚动删除。

> 需 feapder >= 1.6.6

## 配置

在项目 `setting.py` 中配置 InfluxDB 连接信息：

```python
INFLUXDB_HOST = "your-influxdb-host"
INFLUXDB_PORT = 8086
INFLUXDB_UDP_PORT = 8089
INFLUXDB_USER = "your-user"
INFLUXDB_PASSWORD = "your-password"
INFLUXDB_DATABASE = "feapder"
# INFLUXDB_MEASUREMENT = "RedditSpider"  # 可选，不填则自动使用爬虫类名
METRICS_OTHER_ARGS = dict(retention_policy_duration="180d", emit_interval=60)
```

配置完整后，Spider / AirSpider 启动时会自动初始化打点系统，无需手动调用 `metrics.init()` 和 `metrics.close()`。

### measurement 命名

- 默认使用爬虫类名，如 `RedditSpider`
- 可在 `setting.py` 中通过 `INFLUXDB_MEASUREMENT` 显式指定
- 也可通过环境变量 `INFLUXDB_MEASUREMENT` 覆盖

## 内置监控指标

框架自动上报以下指标：

| classify | 说明 |
|----------|------|
| `document` | 各 parser 下载总量/成功/异常/解析异常 |
| 表名 | 入库字段非空情况 + `total count`（需 `yield item` 入库） |
| `proxy` | 代理总量、使用次数、失效数 |
| `users_{status}` | 账号池状态分布 |

- 数据监控图表 title 为表名，折线展示每个字段是否有值及数据总量
- document 为下载情况监控

## Grafana 可视化

1. 在 Grafana 中添加 InfluxDB 1.x 数据源（InfluxQL）
2. 导入项目 [`monitoring/grafana/dashboards/feapder-overview.json`](../../monitoring/grafana/dashboards/feapder-overview.json)
3. 在仪表盘顶部选择对应爬虫 measurement

详细步骤见 [monitoring/grafana/README.md](../../monitoring/grafana/README.md)。

## 自定义监控

```python
from feapder.utils import metrics

metrics.init(
    influxdb_host="localhost",
    influxdb_port=8086,
    influxdb_udp_port=8089,
    influxdb_database="feapder",
    influxdb_user="user",
    influxdb_password="password",
    influxdb_measurement="test_metrics",
)

metrics.emit_counter("key", count=1, classify="test")
metrics.close()
```

参数说明：

```
metrics.emit_counter("key", count=1, classify="test")
```

- `key` 对应折线名称
- `count` 对应数值
- `classify` 对应图表标题（写入 tag `_classify`）

同一 `classify` 下多个 `key` 会出现在同一张图表中：

```python
metrics.emit_counter("key", count=1, classify="test")
metrics.emit_counter("key2", count=1, classify="test")
metrics.emit_counter("哈哈", count=1, classify="test2")
```

## 在 feapder 爬虫中使用

```python
import feapder
from feapder.utils import metrics


class TestSpider(feapder.AirSpider):
    def start_requests(self):
        yield feapder.Request("https://www.baidu.com")

    def parse(self, request, response):
        metrics.emit_counter("success", count=1, classify="自定义的监控指标")


if __name__ == "__main__":
    TestSpider().start()
```

只需 `import metrics` 并调用 `metrics.emit_counter`，框架会自动处理 `init` 与 `close`。在 scrapy 或其他独立脚本中使用时，需手动调用 `metrics.init()` 和 `metrics.close()`。
